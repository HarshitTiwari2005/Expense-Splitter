# Student Expense Splitter (PHP + MySQL, XAMPP)

A visually modern, responsive, and animated ** Expense Splitter** built with **PHP 8**, **MySQL**, and **Bootstrap 5**.
It lets you create groups, add participants, record expenses, auto-calculate who owes whom, and share/export results.

## ✨ Highlights
- Clean modern UI (Bootstrap 5, Google Fonts, AOS animations)
- Add groups, participants, and expenses
- Auto-settlement calculation (server-side PHP)
- Confetti celebration on calculation
- Responsive and mobile-friendly

---

## 🧱 Requirements
- XAMPP (PHP 8+, MySQL/MariaDB, Apache)
- Internet access for CDN assets (Bootstrap, AOS, jsPDF, confetti). You can later download and host them locally if needed.

---


TO run http://localhost/expense-splitter/public


## 🚀 How to Use
1. Click **Create Group**, give it a name.
2. Open the group → **Add Participants** (add all names first).
3. Add expenses (select **who paid**, enter amount & description).
4. Hit **Calculate Settlements** → You’ll see `"A gives ₹X to B"` lines.
5. Share on WhatsApp or **Export PDF**.

---

## 🧪 Tips for Demo 
- Add a few participants & 3–5 expenses to show non-trivial splits.
- After calculating, hit **Export PDF** and **Share on WhatsApp**.
- Mention: *server-side calculation ensures fairness & accuracy*.
- UI extras: pastel name badges, subtle animations, and confetti 🎉
